# ドドンコ農園 (Dodonko Farm) Website

こだわりの有機野菜を届ける「ドドンコ農園」の公式ウェブサイトです。

## プロジェクト概要
自然との共生、土作りへのこだわりを伝えるためのブランドサイト。
視覚的な豊かさと、季節感を感じられるデザインを目指しました。

## ディレクトリ構成
```
dodonko_farm_project/
├── index.html       # トップページ
├── news_harvest.html # 秋の収穫祭レポート
├── news_winter.html  # 冬野菜販売のお知らせ
├── philosophy.html  # こだわり（土作り・環境）
├── products.html    # 商品紹介（季節の野菜）
├── about.html       # 会社概要・お問い合わせ
├── css/
│   └── style.css    # 全体のスタイル定義
├── js/
│   └── main.js      # スムーズスクロール等の動的機能
└── assets/
    └── images/      # 使用画像
```

## 技術スタック
- HTML5
- CSS3 (Vanilla CSS, Responsive Design)
- JavaScript (Vanilla JS)
